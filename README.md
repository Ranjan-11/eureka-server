# Eureka Server

## Description
Eureka Server is the service registry in this microservices architecture.
All microservices, API Gateway, and Admin Server register themselves here
to enable dynamic service discovery and load balancing.

## Tech Stack
- Java
- Spring Boot
- Spring Cloud Netflix Eureka

## Port
8761

## How to Run
mvn spring-boot:run

## Dashboard
http://localhost:8761
